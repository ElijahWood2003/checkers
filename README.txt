	---Written by Elijah Wood in 2019---
Playable checkers written for my APCS A final project. This was originally written in BlueJ.

Functionality:
- Utilizes written text to make moves in the terminal
- Prints each new board state
- Kings can be created to move omni-directionally
- Determines the winner based on the board state
